# Template de cadastro de doadora

Coloque o arquivo oficial nesta pasta com o nome:

`donator-registration-template.docx`

Use placeholders no formato do `docxtemplater`, por exemplo:

`{name}`, `{phone}`, `{birthDate}`, `{homeCollectionYes}`.

Para checkboxes, use placeholders que retornam `X` quando marcados e vazio quando nao marcados.
